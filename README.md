# visual-cortex-research-2024
A collective of my research on the functional connectivity of the visual cortex, specifically the parahippocampal place area (PPA)'s role in visual processing. I include the completed paper, attached figures, and MATLAB code. I started and completed this project in 2024, with supervision from Dr. Joonkoo Park from the University of Massachussetts, Amherst. 

Specifics of each folder:

In "Papers", I have included the full file of my (unpublished) paper, as well as an accompanying literature review. Both were written independently by me, with edits made by Dr. Joonkoo Park.
*note: the file may be too large to open in github. Download the pdf to open on a local browser.

In "Figures", I have included all figures from my paper as their own .png file.

In "Data", I have added a slideshow that compiles the results from all graph theory metrics and statistical tests ran on MATLAB. I have also added a spreadsheet named stimuli_annotation, which is the StudyForrest dataset I used to gather the stimuli for my research. 

In "MATLAB code examples", I have added two instances of code that I wrote and ran on MATLAB as part of my research. 
  The first runs a nonparametric statistical test on characteristic path lengths derived from fMRI data.
  The second separates BOLD signals from fMRI data by scene type (indoor and outdoor).
